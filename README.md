# arelion.dev

Landing page for **Arelion** — a one-person boutique tech studio.

Forked from [antonin.cool](https://github.com/antonhansel/portfolio), rebranded for the company. Same Gatsby setup, blog content removed.

## Develop

```sh
npm install --legacy-peer-deps
npm run develop
```

http://localhost:8000

## Deploy

Auto-deploys to GitHub Pages on push to `main` (`.github/workflows/deploy.yml`). Custom domain via `static/CNAME`.
